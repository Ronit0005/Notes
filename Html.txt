HTML stands for : Hyper text markup language.
* Html is used to structure a web page and its content.
* The component use to design the structure of the website is called html tags.

* Html tags: A container for some paragram or for some other container.
   <p>This is paragraph</p>

* HTML tag is parent of head and body tag.
